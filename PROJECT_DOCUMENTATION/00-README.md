## Quick Start

1. Read **01-OVERVIEW.md** for context
2. Follow **02-DATABASE-SETUP.md** to configure Supabase
3. Create `.env` with `VITE_SUPABASE_URL` and `VITE_SUPABASE_ANON_KEY`
4. Run `npm install` and `npm run dev`
5. Use **03-PAGE-BY-PAGE-EXPLANATION.md** to understand each screen
