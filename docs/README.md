# Documentation

This folder contains all setup and troubleshooting documentation for the Student Dropout Prediction System.

## Files Overview

- **CHECK_SETUP.md** - Quick checklist to verify your setup
- **CREATE_TABLES.md** - Step-by-step guide to create database tables
- **DATABASE_SETUP_INSTRUCTIONS.md** - Detailed database setup instructions
- **DATABASE_FETCHING_FIX.md** - Solutions for database fetching issues
- **DATABASE_FIXES_SUMMARY.md** - Summary of database fixes
- **FIX_ENV_VARIABLES.md** - Guide to fix environment variable issues
- **QUICK_FIX.md** - Quick troubleshooting guide
- **QUICK_START.md** - Quick start guide for new users
- **RESTART_SERVER.md** - Instructions for restarting the development server
- **SETUP.md** - Complete setup guide
- **SOLUTION_GUIDE.md** - Comprehensive solution guide
- **TROUBLESHOOTING.md** - Detailed troubleshooting guide

## Quick Start

1. Read **QUICK_START.md** for a quick overview
2. Follow **SETUP.md** for complete setup instructions
3. Use **CREATE_TABLES.md** to set up your database
4. Refer to **TROUBLESHOOTING.md** if you encounter issues

## Main Project Files

- **create-all-tables.sql** - Single SQL file to create all required tables
- **supabase-schema.sql** - Database schema definition
- **README.md** (in root) - Main project README
